# tms-min-side-topp
Mikrofrontend som trekkes inn av Min Side.

## Kom i gang
1. Bygg tms-min-side-topp ved å kjøre npm run build
2. Start appen lokalt ved å kjøre npm start
3. Appen nås på http://localhost:4100
